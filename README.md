# Measure-Theoretic-Probability

## 📖 Overview

This project develops the foundations of **measure-theoretic probability**, beginning with the construction of measures and integration before applying these tools to modern probability theory.

Rather than treating probability as a collection of formulas, the project builds the subject from first principles using **σ-fields, measures, measurable functions, and the Lebesgue integral**, providing a unified framework for both discrete and continuous random variables.

The project concludes with a rigorous development of the **Lindeberg Central Limit Theorem**, together with numerical simulations in **R** illustrating convergence to the normal distribution.

# Reflection on Measure Theory

The dissertation begins by developing modern measure theory from first principles.

Topics include

- σ-fields and measurable spaces
- Measures and pre-measures
- Carathéodory Extension Theorem
- Construction of the Lebesgue measure
- Measure completion
- Non-measurable sets
- Sets of measure zero
- The Cantor set

The project demonstrates how these concepts provide the rigorous mathematical foundation required for modern analysis and probability.

---

# Reflection on Integration and Probability

Building upon measure theory, the dissertation develops the Lebesgue integral and introduces probability in a measure-theoretic framework.

Major topics include

- Simple and measurable functions
- Lebesgue integration
- Random variables as measurable functions
- Probability measures
- Image (pushforward) measures
- Change of variables theorem
- Product measures
- Fubini–Tonelli theorem
- Radon–Nikodym theorem

Several fundamental convergence results are proved, including

- Monotone Convergence Theorem
- Fatou's Lemma
- Dominated Convergence Theorem

These theorems provide the theoretical tools required to interchange limits and integration throughout probability theory.

---

# Reflection on Probability Theory

The final part of the dissertation applies the measure-theoretic framework to probability.

Topics include

- Joint distributions
- Convolution of probability distributions
- Characteristic functions
- Weak convergence
- Lindeberg Central Limit Theorem

To complement the theoretical development, **R simulations** were used to illustrate convergence predicted by the Lindeberg CLT.

---

# Key Results

The project demonstrates how measure theory unifies discrete and continuous probability within a single mathematical framework.

Key achievements include

- Construction of Lebesgue measure from first principles
- Rigorous development of Lebesgue integration
- Proofs of the major convergence theorems
- Development of image measures and change of variables
- Proof and interpretation of the Lindeberg Central Limit Theorem
- Numerical verification of asymptotic normality using R simulations
